## 登记规则

在架构设计阶段（Phase 3）发现跨模块共享功能时：

1. **在 architecture.md 中标记**: `⚠️ Phase N Dependency: [功能描述]`
2. **在此表登记**:
   - 分配递增编号 PH-XXX
   - 记录涉及模块、临时方案、统一开发时机
3. **实现临时版**: 在各模块中实现最简 Placeholder
4. **定期审查**: 使用 `track-placeholders` MCP 工具扫描遗漏
